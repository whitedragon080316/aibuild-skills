# AI 文案工具人

用 AI 產出課綱、直播講綱、追單文案、直播後內容。學員只需要回答問題，Claude Code 全自動產出。

## 使用方式

`/ai-writer 課綱` — 產出完整課程架構
`/ai-writer 講綱` — 產出 80 分鐘直播講綱
`/ai-writer 追單` — 產出 3 分眾 x 6 波追單文案
`/ai-writer 內容` — 直播後自動產出金句 + 推播 + 短影音腳本

---

## 執行流程（Claude Code 自動處理）

### 1. 課綱生成

**問學員（一次一個）：**
1. 你的課程主題是什麼？（一句話描述）
2. 誰要學？（目標受眾）
3. 學完後學員能做到什麼？

**Claude Code 自動做：**
- 根據回答產出完整 course.json（階段 + 單元 + 描述）
- 每個階段分「策略」+「AI 建置」雙軌
- 建議時長
- 直接寫入專案目錄

**Prompt 規則：**
- 分 4-8 個階段，每階段 4-8 個單元
- 策略類教觀念和方法論，AI 建置類教實際操作
- 順序是學員的學習路徑，前面學的後面會用到

---

### 2. 直播講綱

**問學員（一次一個）：**
1. 你的課程名稱和價格？
2. 目標受眾最痛的 2-3 個點是什麼？
3. 你跟競品最大的差異是什麼？

**Claude Code 自動做：**
- 產出 80 分鐘直播完整大綱
- 每頁投影片內容建議
- 講師備忘錄（提詞）

**結構（不能改的架構）：**

| 時間 | 段落 | 目的 |
|------|------|------|
| 0-5 min | 開場破冰 | 親和力 |
| 5-20 min | 痛點共鳴 | 「他在說我」 |
| 20 min | 第一次互動 | 「打 1」 |
| 20-45 min | 方案展示 | 你的解法 |
| 45 min | 第二次互動 | 「哪個最驚訝」 |
| 45-55 min | 見證案例 | 社會證明 |
| 55-70 min | 報價錨定 | 先讓觀眾算，不直接報 |
| 70 min | 打 777 | **報價前喊**，完課的人才看到價格 |
| 70-80 min | Q&A + CTA | 最後推一波 |

**關鍵規則：**
- 直播控制在 80 分鐘，太長人走
- 每 10 分鐘互動一次
- 777 在報價**之前**喊
- Q&A 不超過 10 分鐘
- 給乾貨但不給全部（給一半留一半）

---

### 3. 追單文案

**問學員（一次一個）：**
1. 直播逐字稿或重點摘要在哪？（給路徑或貼上來）
2. 課程售價和優惠資訊？
3. 有學員見證嗎？（有就貼，沒有就跳過）

**Claude Code 自動做：**
- 產出 3 分眾 x 6 波 = 18 則文案
- 直接寫成可用的文案檔案

| 波次 | 時間 | 主題 |
|------|------|------|
| Wave 1 | +2h | 感謝 + 回放連結 |
| Wave 2 | D+1 | 痛點提醒 |
| Wave 3 | D+2 | 學員故事 |
| Wave 4 | D+3 | 早鳥截止 |
| Wave 5 | D+4 | 異議處理 |
| Wave 6 | D+5 | 最後通知 |

3 種受眾：
- **A. 到課未購買** — 強化價值
- **B. 未到課** — 給回放 + 製造錯過感
- **C. 看回放未購買** — 直接講重點

**文案規則：**
- 每則 200-300 字
- 口語化，不要文謅謅
- 不要用「親愛的」開頭
- 每則結尾有一個 CTA（連結或行動）
- Wave 2、3、5 用純文字，其他用 Flex 卡片

---

### 4. 直播後內容工廠

**問學員：**
1. 直播逐字稿在哪？（給路徑或貼上來）

**Claude Code 自動做：**
- 10 句金句（適合社群貼文）
- 3 則推播訊息（LINE 用）
- 5 支短影音腳本（30-60 秒，含 Hook + 內容 + CTA）
- 重點摘要（給沒看直播的人）
- 全部寫入指定目錄
