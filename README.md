# AI 造局術 — Claude Code Skills

一行指令安裝 AI 自動化工具。

## 安裝方式

### 方法一：終端機一行安裝

安裝單一 Skill：
```bash
bash <(curl -s https://raw.githubusercontent.com/whitedragon080316/aibuild-skills/main/install.sh) line-bot
```

安裝全部：
```bash
bash <(curl -s https://raw.githubusercontent.com/whitedragon080316/aibuild-skills/main/install.sh) all
```

### 方法二：在 Claude Code 裡說

```
幫我安裝 line-bot skill，從 https://github.com/whitedragon080316/aibuild-skills 下載
```
